# ¿Qué Haremos Hoy?

App para descubrir actividades según gustos y ubicación.

## Estructura del proyecto

- `src/app/` — pantallas de la aplicación (rutas)
- `src/components/` — piezas visuales reutilizables
- `src/lib/` — conexión a Supabase y otros servicios externos
- `src/types/` — tipos que reflejan las tablas de la base de datos
- `supabase/migrations/` — historial versionado del esquema de base de datos

## Estado actual
